# Actividad de la versión de Workfront Fusion: Semana del 15 de junio de 2026

Esta página describe todas las mejoras realizadas en Adobe Workfront Fusion durante la semana del 15 de junio de 2026.

Para obtener una lista de todos los cambios recientes, consulte [Actividad de la versión de Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Para obtener una lista de las correcciones de errores recientes en Workfront Fusion, consulte la página [Actualizaciones de mantenimiento de Workfront](https://experienceleague.adobe.com/es/docs/workfront-known-issues/releases/current-updates) y busque cualquier actualización etiquetada como Actualización de mantenimiento de Workfront Fusion.

## Hay nuevos puntos finales disponibles para la API de Adobe Workfront Fusion

Para que la API de Adobe Workfront Fusion sea más flexible y potente, hemos añadido el punto final para las siguientes acciones:

* **Escenarios**: enumere escenarios, obtenga escenarios por identificador, clone un escenario, exporte modelos y obtenga dependencias de escenarios
* **Ejecuciones**: enumere la ejecución y obtenga una sola ejecución
* **Conexiones**: enumera conexiones, obtiene una conexión por identificador y enumera escenarios usando una conexión
* **Almacenes de datos**: enumere los almacenes de datos por equipo y obtenga un almacén de datos por identificador
* **Carpetas**: enumera, crea, actualiza y elimina carpetas
* **Webhooks**: enumera los webhooks y obtén un webhook por identificador

Para ver el registro de cambios completo, consulte [Changelog](https://developer.adobe.com/workfront-fusion-apis/getting-started/changelog/) en la documentación para desarrolladores de Workfront Fusion.

## Ya está disponible el conector de revisión y aprobaciones unificadas de Adobe Workfront

Para ampliar la capacidad de Fusion de trabajar con la revisión unificada y las aprobaciones en Workfront, hemos creado el conector de revisión unificada y aprobaciones de Adobe Workfront. Con las revisiones y aprobaciones unificadas de Adobe Workfront:

* Crear, actualizar o eliminar aprobaciones y etapas
* Adición o recordatorio de participantes de aprobación
* Tomar o eliminar decisiones
* Recuperar información sobre aprobaciones, plantillas o bots de aprobación
* Obtener aprobaciones o participantes sugeridos
* Realizar una solicitud de API personalizada

Para obtener información e instrucciones, consulte [Módulos unificados de revisión y aprobaciones de Adobe Workfront](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/workfront-review-and-approvals-modules.md).

## Nuevos módulos para el conector de Adobe Photoshop

Para expandir las capacidades del conector de Adobe Photoshop, hemos añadido estos módulos:

* Creación de una mesa de trabajo
* Crear o editar un compuesto
* Edición de una imagen con varios ajustes
* Ejecutar acciones, scripts y transformaciones de Photoshop
* Generar un manifiesto

Para obtener información e instrucciones sobre los nuevos módulos, consulte [módulos de Adobe Photoshop](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/adobe-photoshop-modules.md).

## Nuevos módulos disponibles para el conector de Adobe InDesign

Para expandir las capacidades del conector de Adobe InDesign, hemos añadido estos módulos:

* Enviar un script personalizado
* Actualizar versión de aplicación de script personalizada
* Obtener versiones actuales de la aplicación
* Convertir documento de PDF a InDesign

Para obtener información e instrucciones sobre los nuevos módulos, consulte [módulos de InDesign](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/indesign-modules.md).
