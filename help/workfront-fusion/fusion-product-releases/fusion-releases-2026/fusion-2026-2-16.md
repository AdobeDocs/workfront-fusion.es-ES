# Actividad de la versión de Workfront Fusion: semana del martes, 16 de febrero de 2026

En esta página se describen todas las mejoras realizadas en Adobe Workfront Fusion durante la semana del martes, 16 de febrero de 2026.

Para obtener una lista de todos los cambios recientes, consulte [Actividad de la versión de Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Para obtener una lista de las correcciones de errores recientes en Workfront Fusion, consulte la página [Actualizaciones de mantenimiento de Workfront](https://experienceleague.adobe.com/es/docs/workfront-known-issues/releases/current-updates) y busque cualquier actualización etiquetada como Actualización de mantenimiento de Workfront Fusion.

## Agregar peticiones de datos de IA a los escenarios con el nuevo módulo del agente de MCP

Puede incluir una petición de datos de IA en su escenario mediante el Protocolo de contexto de modelo (MCP) combinado con modelos de lenguaje de gran tamaño (LLM). Al configurarlos en el módulo del agente de MCP, puede utilizar la inteligencia artificial para configurar flujos de trabajo eficientes, seguros y flexibles.

MCP le permite utilizar una petición de datos de IA para conectarse con aplicaciones. Dado que el mensaje puede tener en cuenta las variaciones y el contexto, esto reduce la necesidad de configurar manualmente algunos flujos de trabajo, rutas o secciones de escenarios en Fusion.

Para obtener más información sobre el uso de las indicaciones de IA en Fusion, consulte [Agregar una petición de IA a su escenario](/help/workfront-fusion/create-scenarios/add-modules/add-an-ai-prompt-to-your-scenario.md).

Para obtener información sobre la configuración del módulo del agente MCP, consulte [Módulo del agente MCP](/help/workfront-fusion/references/apps-and-modules/tools-and-transformers/model-context-protocol-mcp-connector.md).

## La API de operaciones ya está disponible

Hemos hecho posible obtener datos de operaciones a través de la API de Workfront Fusion. Ahora, puede recuperar datos de operaciones detallados y recuentos de operaciones para sus organizaciones, y agrupar y filtrar resultados por una variedad de parámetros.

Para obtener más información, consulte [API de Workfront Fusion](https://developer.adobe.com/workfront-fusion-apis/#)

## Vea las métricas de rendimiento de toda la organización en Fusion Performance Dashboard

Para facilitar la comprensión del rendimiento de Fusion en toda su organización, hemos creado el nuevo panel de rendimiento. Ahora, los administradores de Fusion pueden ver rápidamente qué escenarios se están ejecutando más, dónde se están produciendo los retrasos y con qué eficacia están funcionando los grupos de trabajo. El panel de rendimiento proporciona visibilidad en tiempo real de los volúmenes de ejecución, la profundidad de cola, el uso del grupo y el rendimiento en el nivel de escenario.

Para obtener información sobre el panel de rendimiento de Fusion, consulte [Ver el panel de rendimiento de una organización](/help/workfront-fusion/set-up-and-manage-workfront-fusion/set-up-and-manage-orgs-and-teams/set-up-orgs-teams-and-users/view-performance-dashboard.md).
