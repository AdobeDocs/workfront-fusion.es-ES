# Actividad de la versión de Workfront Fusion: semana del martes, 26 de enero de 2026

En esta página se describen todas las mejoras realizadas en Adobe Workfront Fusion durante la semana del martes, 26 de enero de 2026.

Para obtener una lista de todos los cambios recientes, consulte [Actividad de la versión de Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Para obtener una lista de las correcciones de errores recientes en Workfront Fusion, consulte la página [Actualizaciones de mantenimiento de Workfront](https://experienceleague.adobe.com/es/docs/workfront-known-issues/releases/current-updates) y busque cualquier actualización etiquetada como Actualización de mantenimiento de Workfront Fusion.

## La API de Workfront Fusion ya está disponible

Para permitirle interactuar mediante programación con Fusion para crear, administrar y operar automatizaciones a escala, hemos lanzado las API de Workfront Fusion. Ahora puede utilizar las API para realizar algunas de las mismas acciones disponibles en la interfaz de usuario de Fusion.

Las API se están lanzando gradualmente y actualmente están marcadas como experimentales. El equipo de Fusion está desarrollando y expandiendo activamente las API, y las actualizaciones se publicarán a medida que haya nueva funcionalidad disponible.

Para obtener más información y documentación sobre la API de Workfront Fusion, consulte [API de Workfront Fusion](https://developer.adobe.com/workfront-fusion-apis/).
