# Actividad de la versión de Workfront Fusion: semana del martes, 12 de enero de 2026

En esta página se describen todas las mejoras realizadas en Adobe Workfront Fusion durante la semana del martes, 12 de enero de 2026.

Para obtener una lista de todos los cambios recientes, consulte [Actividad de la versión de Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Para obtener una lista de las correcciones de errores recientes en Workfront Fusion, consulte la página [Actualizaciones de mantenimiento de Workfront](https://experienceleague.adobe.com/en/docs/workfront-known-issues/releases/current-updates) y busque cualquier actualización etiquetada como Actualización de mantenimiento de Workfront Fusion.

## Ver un escenario y el estado de reintento mientras se procesa la ejecución

Ahora puede ver un escenario mientras los datos se siguen procesando. Si abre un escenario que se está procesando, ya no se le redirigirá fuera del escenario. En su lugar, el escenario se muestra atenuado y solo se muestra un historial parcial hasta que se completa el procesamiento.

Además, ahora se muestra un recuento de reintentos para los escenarios que están reintentando el procesamiento. El recuento va del 1 al 10, y el temporizador de reintentos se duplica cada vez que se reintenta. Si los datos siguen sin estar disponibles después de 5-10 minutos, se mostrará un mensaje para ver los datos más adelante.

## Conexión de servidor a servidor ahora disponible para el conector de Workfront Planning

Se ha agregado la opción de crear conexiones de servidor a servidor para los módulos de Workfront Planning.

Si ya tiene credenciales de servidor a servidor configuradas para una conexión de Workfront, puede usarlas para una conexión de Workfront Planning.

Para obtener información sobre el conector de Workfront Planning, incluido cómo configurar una conexión servidor a servidor, consulte [Módulos de Workfront Planning](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/workfront-planning-modules.md).
