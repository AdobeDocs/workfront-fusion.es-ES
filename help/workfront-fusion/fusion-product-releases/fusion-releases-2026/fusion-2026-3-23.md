## Haga que un módulo personalizado de llamada de API (sondeo) ya esté disponible en el conector de Adobe Authenticator

Para permitir repetir las llamadas de API hasta que se cumpla una condición o límite determinados, hemos creado el módulo Realizar una llamada de API personalizada (sondeo). Este módulo incluye opciones que le permiten configurar llamadas repetidas a la API. Puede hacer que la llamada de API se ejecute hasta que se cumpla una condición concreta, definir un número específico de reintentos y configurar el tiempo entre reintentos.

Para obtener más información, consulte [Realizar una llamada de API personalizada (sondeo)](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/adobe-authenticator-modules.md#make-a-custom-api-call-polling) en el artículo Módulos de Adobe Authenticator.
