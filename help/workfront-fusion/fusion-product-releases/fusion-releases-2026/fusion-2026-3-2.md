# Actividad de la versión de Workfront Fusion: semana del martes, 02 de marzo de 2026

En esta página se describen todas las mejoras realizadas en Adobe Workfront Fusion durante la semana del martes, 02 de marzo de 2026.

Para obtener una lista de todos los cambios recientes, consulte [Actividad de la versión de Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Para obtener una lista de las correcciones de errores recientes en Workfront Fusion, consulte la página [Actualizaciones de mantenimiento de Workfront](https://experienceleague.adobe.com/es/docs/workfront-known-issues/releases/current-updates) y busque cualquier actualización etiquetada como Actualización de mantenimiento de Workfront Fusion.

## Crear funciones de JavaScript personalizadas para usarlas en escenarios

Para proporcionar una mejor flexibilidad y control de sus escenarios, hemos agregado la capacidad de crear una biblioteca de funciones de JavaScript personalizadas que puede utilizar en escenarios. Puede crear funciones personalizadas en el área Funciones de Fusion. A continuación, agregue estas funciones a sus escenarios en forma de módulo de Adobe App Builder.

Dado que las funciones personalizadas funcionan a través de Adobe App Builder, su organización debe tener una licencia de Adobe App Builder para utilizarlas.

Para obtener más información sobre el uso de funciones personalizadas en Fusion, consulte [Asignar datos con funciones personalizadas](/help/workfront-fusion/create-scenarios/map-data/map-using-custom-functions.md).

## La autorización del token de acceso personal ya está disponible para el conector Airtable

Para que el conector Airtable sea más flexible, hemos agregado la capacidad de usar tokens de acceso personal (PAT) para la autorización. Ahora, al crear una conexión, puede elegir utilizar la autorización de OAuth2 o PAT.

Para obtener información e instrucciones, consulte [Módulos airtable](/help/workfront-fusion/references/apps-and-modules/third-party-connectors/airtable-modules.md).
