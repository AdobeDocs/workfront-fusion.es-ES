# Actividad de la versión de Workfront Fusion: Semana del 27 de abril de 2026

Esta página describe todas las mejoras realizadas en Adobe Workfront Fusion durante la semana del 27 de abril de 2026.

Para obtener una lista de todos los cambios recientes, consulte [Actividad de la versión de Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Para obtener una lista de las correcciones de errores recientes en Workfront Fusion, consulte la página [Actualizaciones de mantenimiento de Workfront](https://experienceleague.adobe.com/es/docs/workfront-known-issues/releases/current-updates) y busque cualquier actualización etiquetada como Actualización de mantenimiento de Workfront Fusion.

## Nuevos módulos disponibles para el conector Veeva Vault

Para ampliar las capacidades del conector Veeva Vault, hemos añadido los siguientes módulos nuevos:

* Extraer varios archivos
* Recuperar resultados de extracción
* Cargar varios archivos
* Recuperar resultados de carga
* Enumerar elementos en una ruta

Para obtener información sobre los módulos de Veeva Vault, incluidos los nuevos módulos, consulte [Módulos de Veeva Vault](/help/workfront-fusion/references/apps-and-modules/third-party-connectors/veeva-vault-modules.md).

## Editar webhooks para Workfront y Workfront Planning

Para facilitar la actualización de los webhooks para los conectores de Workfront y Workfront Planning, hemos añadido la capacidad de editar los webhooks existentes. Ahora, puede actualizar los webhooks desde el módulo del webhook o desde la lista de Webhooks.

Anteriormente, para actualizar un webhook era necesario crear un nuevo webhook y asignarlo a escenarios manualmente.

Tenga en cuenta que un gancho web actualizado se considera una nueva suscripción de evento.

Para obtener información sobre cómo actualizar los webhooks, incluidas las consideraciones sobre la nueva suscripción de evento, consulte [Editar webhooks](/help/workfront-fusion/manage-scenarios/edit-webhooks.md).

## Añadir una descripción a un escenario

Para facilitar la identificación de escenarios en el área de Escenarios, se ha añadido la capacidad de agregar una descripción.

Las descripciones se pueden añadir en la Configuración de escenario o en el área de Escenarios y están limitadas a 240 caracteres.

Para obtener más información acerca de la configuración de escenarios, vea [Configurar escenarios](/help/workfront-fusion/create-scenarios/config-scenarios-settings/configure-scenario-settings.md).

Para obtener más información sobre las acciones disponibles en el área de Escenarios, consulte [Administrar escenarios desde el área de Escenarios](/help/workfront-fusion/manage-scenarios/manage-scenarios-from-list.md).

## Ver el historial de ejecución de una cadena de escenarios

Para facilitar la depuración o el seguimiento de datos a través de escenarios encadenados, hemos añadido la capacidad de ver el historial de escenarios de la cadena. Ahora, en el área Relaciones entre escenarios, puede ver el historial de ejecución. Al hacer clic en una ejecución específica se muestran los datos pasados entre escenarios para esa ejecución.

Para obtener más información sobre cómo encadenar escenarios, vea [Encadenar varios escenarios](/help/workfront-fusion/create-scenarios/plan-a-scenario/chain-scenarios.md).

Para obtener más información sobre el área Relaciones entre escenarios encadenados, vea [Ver y administrar relaciones entre escenarios encadenados](/help/workfront-fusion/manage-scenarios/view-chained-scenario-relationships.md).
